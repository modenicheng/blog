### 自增自减

```c
// i = i + 1
i++; 
++i;

// i = i - 1
i--; 
--i;
```

::: tip `++i` 和 `i++` 的区别

- `++i` 先自增再使用
- `i++` 先使用再自增

例子：

```c
int i = 3;
printf("%d\n", ++i); // 4
printf("%d\n", i++); // 4
printf("%d\n", i); // 5
```

:::

### 符合运算符

```c
// i = i + 5
i += 5;

// i = i - 5
i -= 5;

// i = i * 5
i *= 5;

// i = i / 5
i /= 5;
```
