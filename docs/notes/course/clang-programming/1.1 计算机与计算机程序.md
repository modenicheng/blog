## 计算机的组成

计算机由硬件和软件组成，硬件是计算机的实体，软件是计算机的指令。

### 硬件

硬件是计算机的实体，包括：

- 中央处理器（CPU）：计算机的大脑，负责执行指令。
- 内存（RAM）：计算机的临时存储器，用于存储正在执行的程序和数据。
- 输入设备：用于向计算机输入数据，如键盘、鼠标等。
- 输出设备：用于向计算机输出数据，如显示器、打印机等。
- 存储设备：用于长期存储计算机程序和数据，如硬盘、光盘等。

### 软件

软件是计算机的指令，包括：

- 操作系统：管理计算机硬件和软件资源，提供用户界面。
- 应用程序：用于完成特定任务的程序，如文字处理、图像处理等。
- 编程语言：用于编写计算机程序的编程语言，如C、Java、Python等。
