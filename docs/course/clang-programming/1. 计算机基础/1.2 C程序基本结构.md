## 第一声问候：Hello World

```c
/* HELLO WORLD！*/
#include <stdio.h>

int main()
{
    printf("Hello World");
    return 0;
}
```

- C 程序
  - 函数构成，从 `main` 函数开始执行

- 基本字符集：ASCII，注释无限制
  - <LinkCard title="ASCII 维基百科" href="https://en.wikipedia.org/wiki/ASCII" />
- 标识符
  - 由字母、下划线开头
  - 由字母、数字或下划线组成的字母数字串
  - 一般有一些含义
  - 可以作为变量名、常量名、数组名、函数名类型名等
- 关键字（保留字，reserved word）
- 语句
  - 由标识符、关键字等构成的程序中的最小运行单元
  - 如：`printf("Hello World");`
- 函数
  - 由一条或多条语句构成的完成特定功能的独立程序模块
  - 库函数
  - 自定义函数
- ~~表达式（？）~~

## 程序编写准则

1. 正确性
2. 可读性
3. 健壮性
4. 高效率、低存储量（时间、空间上均需优化）
